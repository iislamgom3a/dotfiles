## Install Packages 

1. clone the repo to your home dir

```sh 
git clone https://github.com/iislamgom3a/dotfiles ~/
```

2. Navigate to the dir 

```sh
cd ~/dotfiles
```

3. change script persmission

```sh
chmox +x ./install_packages.sh
```

4. run the script

```sh
./install_packages.sh
```
