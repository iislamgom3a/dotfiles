# Fedora Development Environment Setup

- This repository contains a setup script and configuration files for a  dev environment on Fedora Linux.
- tested for fedora Fedora Linux 43 (Workstation Edition)

## Installatoin

```bash
# Clone this repository
git clone <your-repo-url> ~/dotfiles
cd ~/dotfiles

# Make the setup script executable
chmod +x install.sh

# Run the installation script
./install.sh
```
